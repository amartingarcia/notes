SOPS¶

SOPS is a tool that allows the encryption of local files. For our use case we use a Key from Key Vault declared in the .sops.yaml file for file encryption.

SOPS Commands¶

Show file¶

$ cat file.yaml
---
secrets:
    admin-creds:
        stringData:
            user: user-fake
            password: password-fake

$ sops -e -i file.yaml
